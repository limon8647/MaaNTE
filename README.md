MaaNTE helps you manage routine tasks in Neverness to Everness. The software uses the MAAFramework to handle repetitive actions. You save time while the program plays the game for you.

## 🛠 Features

This program automates daily game chores. It manages the following tasks:

*   **Fishing:** The system detects fish, catches them, and sells the catch automatically.
*   **Bait Management:** The program buys bait when your supply runs low.
*   **Coffee Shop:** It prepares coffee and collects your revenue without your input.
*   **Dialogue:** The system skips story dialogue to move you through content fast.
*   **Daily Rewards:** It logs in and claims your daily gifts from the menu.

## ❓ Frequently Asked Questions

**Is this program safe to use?**
The software functions as an overlay. It reads the screen to make decisions based on your game state. It does not modify game files or inject code into the game process.

**What do I do if the program stops working?**
Restart both the game and the MaaNTE program. If the issue remains, ensure your screen resolution matches the game settings.

**Does this program work on Mac or Linux?**
Currently, this tool supports Windows only. 

**Can I run custom tasks?**
The program follows a set design based on the MAAFramework. You can select features from the interface, but you cannot create custom scripts at this time.

**How do I report a bug?**
You may reach out through the QQ group associated with the project. Provide a screenshot of the issue to help the developers fix the error.
